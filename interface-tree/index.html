    ChatTea иҮӘжңүе‘Ҫд»Өдҝқз•ҷдёәпјҡ

    • set-tokenпјҡжҠҠе·Іжңү Gitea д»ӨзүҢ й…ҚзҪ®еҲ° ChatEnv е’Ң д»“еә“жң¬ең° git configпјӣ
    • serverпјҡз®ЎзҗҶеҶ…йғЁ / иҮӘжүҳз®Ў Gitea зҡ„е®үиЈ…гҖҒapp.iniгҖҒиҝӣзЁӢе’ҢеҒҘеә·жЈҖжҹҘгҖӮ

    иҒҢиҙЈиҫ№з•Ң

    • set-tokenпјҡжҠҠй»ҳи®Ө Gitea base URL е’Ң API д»ӨзүҢ еӯҳе…Ҙ ChatEnv зҡ„ ChatTea active й…ҚзҪ®жЎЈгҖӮ
    • token create/list/delete/bootstrapпјҡйҖҡиҝҮ BasicAuth еҲӣе»әгҖҒеҲ—еҮәгҖҒеҲ йҷӨ Gitea и®ҝй—®д»ӨзүҢпјӣеј•еҜј дјҡеҲӣе»әжҲ–иҪ®жҚўжүҳз®Ў д»ӨзүҢпјҢз„¶еҗҺй…ҚзҪ® ChatTea/Git еҮӯжҚ®гҖӮ
    • auth login/status/tokenпјҡеҗҢдёҖеҘ—еҮӯжҚ®зҠ¶жҖҒдёҠзҡ„иҫ…еҠ©е‘ҪеҗҚз©әй—ҙгҖӮ
    • apiпјҡеҜ№е°ҡжңӘдёҖзӯүе°ҒиЈ…зҡ„ и·Ҝз”ұ жҸҗдҫӣеҺҹе§Ӣ Gitea API passthroughгҖӮ
    • server backup dumpпјҡе°ҒиЈ… Gitea dumpпјҢж”ҜжҢҒе®Ңж•ҙеӨҮд»Ҫе’Ң --database mysql --db-only иҝҒз§» SQL еҜјеҮәгҖӮ
    • server migrate mysqlпјҡйҖҡиҝҮ ChatData-managed MySQL жҠҠеҪ“еүҚ SQLite Gitea backend иҝҒз§»еҲ° MySQLпјҢе№¶еӨҮд»Ҫ/жӣҙж–° app.iniгҖӮ
    • server installпјҡй»ҳи®Өе®үиЈ…жңҖж–° ChatArch еҶ…йғЁ Giteaпјӣ--version еҸҜеӣәе®ҡ еҸ‘еёғзүҲжң¬гҖӮ
    • server initпјҡдёәжң¬ең° SQLite ж”Ҝж’‘зҡ„ Gitea е®һдҫӢеҲӣе»әжңҖе°Ҹ app.iniпјӣзӣ‘еҗ¬ең°еқҖ е’Ң HTTP port жҳҜ CLI init еҸӮж•°пјҢдёҚжҳҜ Env еӯ—ж®өгҖӮ
    • server serveпјҡеүҚеҸ°иҝҗиЎҢ GiteaпјҢз”ЁдәҺи°ғиҜ•жҲ–дёҖж¬ЎжҖ§дјҡиҜқгҖӮ
    • server start/stop/restart/status/logsпјҡз®ЎзҗҶеӣәе®ҡзҡ„ з”ЁжҲ·зә§ systemd жңҚеҠЎ chattea-gitea.serviceгҖӮ
    • server version/healthпјҡжЈҖжҹҘжң¬ең° дәҢиҝӣеҲ¶ж–Үд»¶ жҲ–е·Ій…ҚзҪ®зҡ„ Gitea HTTP з«ҜзӮ№гҖӮ
    • server config path/show/get/setпјҡжҹҘзңӢжҲ–жӣҙж–°жүҳз®Ў Gitea app.iniпјҢдёҺ ChatEnv зӢ¬з«ӢгҖӮ
    • repo list/view/create/edit/generateпјҡиҰҶзӣ–еҹәзЎҖд»“еә“еҲ—иЎЁгҖҒиҜҰжғ…гҖҒеҲӣе»әгҖҒзј–иҫ‘е’Ңд»ҺжЁЎжқҝз”ҹжҲҗпјӣrepo create --template еҸҜеҲӣе»әжЁЎжқҝд»“еә“гҖӮ
    • repo cloneпјҡд»Һе·Ій…ҚзҪ® Gitea е®һдҫӢ clone д»“еә“пјҢдёҚйўқеӨ–й…ҚзҪ® Git йүҙжқғ headerгҖӮ
    • repo migrateпјҡд»Һе·Іжңү Git clone URL еҲӣе»ә Gitea иҝҒз§»гҖӮ
    • issue list/view/create/edit/close/reopen/deleteпјҡйҖҡиҝҮ /repos/{owner}/{repo}/issues и·Ҝз”ұ з®ЎзҗҶд»“еә“ й—®йўҳгҖӮ
    • issue comment list/create/edit/deleteпјҡйҖҡиҝҮ й—®йўҳ иҜ„и®ә и·Ҝз”ұ з®ЎзҗҶиҜ„и®әпјӣPR иҜ„и®әеӨҚз”Ё Gitea й—®йўҳ-иҜ„и®ә жЁЎеһӢгҖӮ
    • issue label add/remove е’Ң issue assign add/removeпјҡйҖҡиҝҮ д»“еә“зә§ й—®йўҳ и·Ҝз”ұ з®ЎзҗҶ ж Үзӯҫ е’Ң иҙҹиҙЈдәәгҖӮ
    • label list/view/create/edit/deleteпјҡйҖҡиҝҮ /repos/{owner}/{repo}/labels з®ЎзҗҶд»“еә“ ж ҮзӯҫгҖӮ
    • milestone list/view/create/edit/close/deleteпјҡйҖҡиҝҮ /repos/{owner}/{repo}/milestones з®ЎзҗҶд»“еә“ йҮҢзЁӢзў‘гҖӮ
    • pr list/view/create/edit/close/reopen/merge/diff/patch/commits/filesпјҡйҖҡиҝҮ /repos/{owner}/{repo}/pulls и·Ҝз”ұ з®ЎзҗҶ PRгҖӮpr checkout дёҚеұһдәҺеҪ“еүҚиғҪеҠӣйқўгҖӮ
    • pr review list/create/submitпјҡйҖҡиҝҮ /repos/{owner}/{repo}/pulls/{index}/reviews и·Ҝз”ұ з®ЎзҗҶ PR reviewгҖӮ
    • release list/view/latest/by-tag/create/edit/deleteпјҡйҖҡиҝҮ /repos/{owner}/{repo}/releases и·Ҝз”ұ з®ЎзҗҶ еҸ‘еёғзүҲжң¬гҖӮ
    • release asset list/deleteпјҡжҹҘзңӢе’ҢеҲ йҷӨ еҸ‘еёғзүҲжң¬ йҷ„д»¶гҖӮMultipart йҷ„д»¶ дёҠдј  зӯү HTTP client ж”ҜжҢҒ дёҠдј  еҗҺеҶҚиЎҘгҖӮ
